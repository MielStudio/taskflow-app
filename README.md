# TaskFlow

TaskFlow is a responsive task management dashboard created as a frontend portfolio project.

The project demonstrates a modern SaaS-style interface with authentication pages, dashboard analytics, task management, task creation, filtering, sorting, responsive layouts, animations, and local data storage.

## Live Demo

https://taskflow-app-five-self.vercel.app/

## GitHub Repository

https://github.com/MielStudio/taskflow-app

## Features

- Landing page
- Login page
- Registration page
- Dashboard
- Tasks list
- Task details page
- Create task page
- Profile page
- Responsive sidebar and mobile navigation
- Task filtering by status, priority, assignee, and due date
- Task sorting by due date, priority, and title
- List and grid task views
- Task creation with LocalStorage
- Search for tasks and users
- Animated UI interactions
- Modal windows for unavailable demo features

## Tech Stack

- React
- Vite
- React Router
- Tailwind CSS
- Framer Motion
- Lucide React
- LocalStorage

## Project Purpose

This project was built as a portfolio showcase to demonstrate frontend development skills, UI/UX implementation, responsive design, routing, state management, and interactive dashboard components.

TaskFlow is not a full production task management system. Some features are intentionally limited and display demo modal windows.

## Main Pages

- `/` — Landing Page
- `/login` — Login
- `/register` — Register
- `/dashboard` — Dashboard
- `/tasks` — Tasks List
- `/tasks/:id` — Task Details
- `/create-task` — Create Task
- `/profile` — Profile
